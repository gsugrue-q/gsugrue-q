# Meet Claudey McClaudeface

<img src="https://github.com/user-attachments/assets/aea97486-8e64-489e-ac03-26e6e27baa3e" width="160" height="160">


Claudey is the first in a planned roster of Claude-powered code review bots — the **generalist** of the bunch. He brings a broad eye to pull requests: catching the obvious stuff, flagging the not-so-obvious stuff, and offering a second perspective on anything that smells off.

The plan is to grow the team. Future bots will be **specialists**, each tuned for a particular domain with their own quirks and pet peeves. The goal isn't just more reviews — it's more *kinds* of reviews. A single default reviewer tends to converge on a predictable set of feedback. A small panel of opinionated specialists, each looking at the same diff through a different lens, breaks out of that determinism and surfaces things a generalist would miss.

Claudey signs off on every review he leaves. If you see the top hat, you know he's been by.
